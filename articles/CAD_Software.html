CAD Software

CAD software is waar het ontwerp werk begint; dankzij de parametrische attributen van deze programmas, kan je op eenderwelk moment heel je ontwerp aanpassen op een fundamenteel level.
Deze software is ook gebaseert op vectoren, waardoor je niet gelimiteerd bent tot polygonen zoals in conventionele 3D software:
Bv. Blender, Maya, 3DS MAX, Houdini, etc.
Daarnaast is er ook CAM software, waarin de ontwerpen omgevormd worden tot 3D paden waarmee de CNC machines gecontroleerd worden.
